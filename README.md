# BuildCon Website

A responsive static landing page for a construction or architecture brand built with HTML, CSS, and Bootstrap.

## Project Overview

This project is a modern marketing website for BuildCon. It includes a responsive navbar, carousel hero section, service cards, portfolio visuals, team highlights, contact details, and an off-canvas contact panel.

## Features

- Responsive layout powered by Bootstrap 5
- Full-height hero carousel with caption overlays
- Sticky navigation bar with smooth section links
- Service cards, portfolio previews, and team section
- Off-canvas contact panel for quick access
- Custom styling in `buildcon.css`
- Google Fonts and Bootstrap Icons integration

## Files

- `buildcon.html` — main website HTML structure
- `buildcon.css` — custom styles and responsive behavior
- `img/` — image assets used in the layout
- `README.md` — project documentation

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (Montserrat)

## Usage

1. Clone or download the repository.
2. Open `buildcon.html` in a web browser.
3. Optionally, serve it from a local web server for faster asset loading.

## Customization

- Edit `buildcon.html` to update section text, navigation links, or image references.
- Update `buildcon.css` to change colors, spacing, hover effects, and responsive behavior.
- Replace images in the `img/` folder with your own photography or illustrations.

## Notes

- The project is fully static and does not require a backend.
- It is ideal for use as a landing page, portfolio demo, or construction company showcase.
